# toned

Monorepo for Toned — lightweight, platform-agnostic styling primitives.

## Packages

- **@toned/core**: core styling primitives and utilities — see [packages/toned-core/README.md](packages/toned-core/README.md)

- **@toned/react**: React (web + native) bindings — see [packages/toned-react/README.md](packages/toned-react/README.md)

## Install

- For projects using the packages: `npm install @toned/core @toned/react`

## Quick start

- React users: follow the examples and setup in [packages/toned-react/README.md](packages/toned-react/README.md).
